# Bee Communications — Design Architecture

## Concept: "Warm Authority"
Premium B2B consultant website. Warm white base with cream/ivory tones. Yellow-orange elevated to sophisticated amber-gold. Oversized serif headlines with tight tracking. Organic honeycomb-inspired shapes as subtle design elements. Balance of editorial structure and human warmth. Professional but approachable.

## Key Design Decisions
- Light mode (warm white #FEFDFB base)
- Less yellow than current site — amber-gold as accent only
- Bee metaphor via subtle patterns, not literal illustrations
- No photos available yet — use abstract shapes + logo elements as hero visuals
- German only (English version deferred)
- Siegel = custom SVG badge designs for certifications
